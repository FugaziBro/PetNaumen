Запускать через класс TestBasic
Тест проверяет что при наведении на пункт меню меняется цвет, затем переходит во вкладку карьера
и отправляет http/get запрос по всем ссылкам в хедере, чтобы проверить, что все они "небитые"
Запросы через этот метод сделаны через несколько потом, что позволило выиграть ~7-10 секунд
